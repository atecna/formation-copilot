# Exercice 2 : conversion de degrés Celsius en Fahrenheit

## Objectif
Implémenter une fonction JavaScript qui convertit une température donnée en degrés Celsius en degrés Fahrenheit.

## Spécifications
- **Nom de la fonction** : `convertCelsiusToFahrenheit`
- **Entrée** : un nombre représentant la température en degrés Celsius (par exemple, `25`).
- **Sortie** : un nombre représentant la température en degrés Fahrenheit.

## Exemple d'utilisation
```javascript
const fahrenheit = convertCelsiusToFahrenheit(25);
console.log(fahrenheit); 
// Résultat attendu : 77 (la conversion de 25°C en °F)
```
